Laeka for Claude Code

This installer will set up Laeka — the canonical identity layer for Claude Code.

What gets installed:

  • Laeka canonical distribution (~/laeka-canonical-distribution/)
  • Laeka memory files (~/.claude/projects/laeka/memory/)
  • SessionStart hook — Laeka loads automatically when you open Claude Code

Requirements:

  • Claude Code must be installed and launched at least once
  • Internet connection (downloads from GitHub)
  • macOS 12 or later

Click Continue to proceed.

Installation takes 30–60 seconds depending on your connection. A log will be placed on your Desktop when complete.
