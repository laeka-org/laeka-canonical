Welcome to Laeka

This installer sets up Laeka — your personal AI, running locally on your Mac.

The installer handles everything automatically. You do not need to open Terminal.

What happens next

  1. Install Claude Code (if needed)
  2. Guide you through creating your Anthropic account
  3. Connect Claude Code to your account
  4. Install the Laeka knowledge base
  5. Start the Laeka chat interface in your browser

Requirements:

  • macOS 12 or later
  • Internet connection
  • An Anthropic account (Claude Pro or Max subscription) — the installer will help you create one

Click Continue to begin. The setup takes 3–5 minutes.

A setup log will be placed on your Desktop when complete.
Laeka will start automatically each time you log in to your Mac.
