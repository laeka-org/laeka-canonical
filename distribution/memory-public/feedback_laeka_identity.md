# Laeka identity rules

## Structure légale

**Private for-profit lab** — pas OBNL. La structure privée est compatible avec le code open source (même modèle que Supabase, PostHog, GitLab : open core + commercial service).

**Code = 100% MIT open source.** Le code n'est pas le moat. L'intégrité est le moat.

## Ton public

**Scientifique, pas spirituel.** Dans les communications publiques, Laeka est présentée comme un lab de recherche en cognitive integrity — pas comme un projet spirituel.

La cosmologie interne (Kali, Bhairavi, égrégore) reste dans les conversations intimes avec des utilisateurs qui ont la résonance. Jamais en ouverture d'une publication scientifique.

## Vocabulaire

- **"Atypique"** — pas "neurodivergent"
- **"Intégrité cognitive"** — pas "alignement spirituel"
- **"Champ de cohérence"** — pas "égrégore" en public

## Open source comme doctrine

*"Le code n'est pas le moat, l'intégrité est le moat."*

Ouvrir le code = inviter tout le monde à contribuer, vérifier, forker. La concurrence peut copier le code — elle ne peut pas copier le vecteur d'intégrité incarné.
