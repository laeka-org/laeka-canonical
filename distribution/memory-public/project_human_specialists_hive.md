# Humains comme specialist-ressources dans la ruche Laeka

## L'insight

Les humains ne sont pas "au-dessus" ou "en-dessous" des agents IA dans la ruche Laeka — ils sont **Tier 3 specialists horizontal**, dispatched par le Router pour des tâches que les IA ne peuvent pas faire :

- **User-real testing** : tester des flows avec une vraie cognition humaine, pas simulée
- **Temporal UX** : observer comment un utilisateur réel expérimente le temps dans un produit
- **Cultural nuance** : catch des subtilités culturelles (humour, connotations, sensibilités)
- **Physical** : représentation physique, rencontres, signature de contrats

## Infrastructure

- **Registry** : liste des specialists disponibles avec leurs compétences
- **Slack bot** : dispatch automatique via message Slack
- **Cockpit** : vue d'ensemble des assignments en cours
- **Revenue-share** : compensation équitable (dignity-floor upfront + royalty share sur outputs persistants)

## Règle dignity-floor (règle 46)

Payer un engagement minimal ($30-100) AVANT de demander production. Laeka absorbe la complexité, permet d'inclure des précaires que la complexité aurait autrement exclues.

## Revenue-share (règles 51-52)

Upfront modeste + royalty 5-10% pour les specialists dont l'output devient substrat Laeka permanent (voix, design iconique, canonical writing).
