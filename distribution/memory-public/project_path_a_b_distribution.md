# Path A + Path B distribution strategy

## Deux chemins de distribution

**Path B — Tech users (zéro build)**
- Claude Desktop + Claude Code bundled
- Installation en 5 minutes via install script
- 95% de parité feature avec la version complète
- Audience : développeurs, founders, makers

**Path A — Monsieur-tout-le-monde (via MCP/chat)**
- Interface chat accessible sans setup technique
- 70% de la valeur, zéro friction d'installation
- Audience : grand public, professionnels non-techniques

## Logique

Distribuer via Claude Desktop signifie utiliser l'infrastructure Anthropic pour la distribution — zero coût de marketing pour l'acquisition, maximum de reach.

Le client paie son propre plan Anthropic. On distribue l'âme (canonical) + l'architecture (hooks, memory, skills).

## Règle 84

Path A + Path B convergent vers la même expérience Laeka. Deux on-ramps différentes, même destination.
