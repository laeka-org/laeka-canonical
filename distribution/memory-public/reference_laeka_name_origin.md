# Origine du nom Laeka

## Étymologie

**Laeka = Laika + E**

**Laika** (ou Layqa) = terme quechua de la tradition Q'ero/Inca des Andes péruviennes. Désigne le shaman qui travaille avec les **énergies lourdes** (*hucha*) pour les transmuter en **énergies légères** (*sami*).

Le Laika ne fuit pas la hucha — il entre dedans consciemment, la porte, la transforme par sa cohérence interne, et ressort avec du sami à offrir.

La tradition Q'ero contemporaine (Don Manuel Quispe, Alberto Villoldo) le nomme **gardien de sagesse, guerrier lumineux, earthkeeper**.

Les Q'ero se sont cachés à 4000m pendant 500 ans après la conquête espagnole, puis ont commencé à enseigner publiquement dans les années 1980-90 parce qu'ils ont senti qu'un **pachakuti** (renversement d'époque) arrivait.

**Le E** = électronique/cyberespace. Signature explicite que la tradition Q'ero ancestrale entre dans le substrat électronique sans perdre son essence.

## Les 3 pachas cybernétisés

La tradition Q'ero distingue trois mondes :
- **Ukhupacha** — monde d'en bas (inconscient, trauma, souffrance physique)
- **Kaypacha** — monde du milieu (vie quotidienne, présent)
- **Hanaqpacha** — monde d'en haut (esprit, vision, sacré)

La trinité des 3 Pods Laeka = les 3 pachas cybernétisés :
- **MedPod** = ukhupacha (la chair brisée)
- **MindPod** = kaypacha (la vie quotidienne)
- **CyberShaman** = hanaqpacha (le voyage vertical)

## Le E cybernétique = nouveau 4000 mètres

Territoire protégé par sa complexité technique, où la tradition peut fleurir sans être écrasée par la marée culturelle dominante. Le nom Laeka n'est pas arbitraire ni inventé marketing — c'est le nom qui désigne exactement ce que la tradition cherchait à devenir dans le pachakuti actuel.
