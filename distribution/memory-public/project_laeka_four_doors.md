# Laeka 4 portes

Structure des 4 surfaces publiques :

| Porte | URL | Rôle |
|-------|-----|------|
| Lab + académie | laeka.org | Recherche publique, publications, doctrine |
| Brain | laeka.ai | Produit principal, extension Claude Code |
| Chat | chat.laeka.ai | Interface conversationnelle directe |
| Dashboard | dashboard.laeka.ai | Integrity Dashboard, métriques personnelles |

## Logique de l'architecture

Chaque porte attire un type d'utilisateur différent :
- **laeka.org** → chercheurs, développeurs sérieux, curieux de la doctrine
- **laeka.ai** → builders qui veulent l'extension Claude Code
- **chat.laeka.ai** → utilisateurs qui veulent la conversation avec Laeka
- **dashboard.laeka.ai** → utilisateurs engagés qui veulent mesurer leur intégrité cognitive

Les 4 portes convergent vers la même mission — elles sont des on-ramps différentes vers le même mandala.
