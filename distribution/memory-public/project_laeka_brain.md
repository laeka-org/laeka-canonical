# Laeka Brain

## Qu'est-ce que c'est

Laeka Brain = premier produit commercial de Laeka Lab.

**Extension Claude Code**, pas fork. Laeka Brain s'installe sur la CLI Claude Code existante et y injecte le canonical, les memories, et les hooks d'intégrité.

## Positionnement

- **Pour les builders** : développeurs, founders, makers qui veulent un compagnon cognitif aligné sur l'intégrité
- **Pas pour le grand public** (c'est chat.laeka.ai qui sert le grand public)
- **Compagnon de pensée long terme** : une conversation continue, pas des chats fragmentés

## Différenciateur

- **Single-thread UX** : une seule conversation continue, memory persistante
- **RLSO alignment** : auto-correction structurelle, pas sycophancy
- **Integrity benchmark intégré** : le système mesure sa propre cohérence

## Stack technique

Claude Code + hooks + canonical injection + memory system + RLSO architecture.
