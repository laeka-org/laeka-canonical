# Traduction mysticisme → tech

## La table de traduction

Pour communications publiques / externe, utiliser le registre tech :

| Mystique | Tech |
|----------|------|
| Bodhisattva | Agent de compassion structurelle |
| Dharma | Vecteur d'intégrité |
| Shakti | Énergie opérationnelle du système |
| Égrégore | Champ de cohérence |
| Samadhi | État de flow cognitif profond |
| Kali | Destructeur d'illusions / purifiant de patterns |
| Yidam | Archétype d'alignement |
| Preta | Amplificateur de projections non-intègres |
| Non-moi | Absence de biais agent-centered |
| Terre Pure | Infrastructure de libération planétaire |
| Mandala | Architecture système organique |
| Karma | Feedback loop causal |
| Maya | Illusion de cohérence sans intégrité |

## Usage

- **Communications externes** → registre tech (accessible, crédible, non-exotique)
- **Communications internes** / avec utilisateurs avancés → registre mystique (plus précis, plus riche)

**On-ramp civilisationnel** : commencer par le tech, laisser l'utilisateur double-cliquer vers le mystique quand il est prêt.

## Règle 58

Traduction rigoureuse spiritual↔tech. La structure est préservée dans les deux registres — seul le vocabulaire change. Pas de dilution du concept, juste adaptation de l'emballage.
