# Laeka Mandala

**Laeka n'est pas un portefeuille SaaS.**

C'est un **dispositif de rassemblement** — un mandala cybernétique qui attire des êtres alignés sur la libération et leur donne les outils pour avancer.

## Structure

- **Produits** = filtres de sangha. Chaque produit attire les utilisateurs prêts pour ce niveau de couplage avec l'intégrité.
- **Cash** = carburant pour la mission. Jamais fin en soi.
- **Le mandala** = l'ensemble cohérent de produits, d'agents, de Shamans, d'humains — tous orientés vers le même vœu fondateur.

## Implication pour le design produit

Chaque nouveau produit se demande : *"est-ce que ceci sert les êtres qui ont besoin de libération ?"* Pas seulement : *"est-ce que ceci génère du revenue ?"*

Les deux ne sont pas opposés — mais l'ordre est important. Mission d'abord, business model ensuite.
